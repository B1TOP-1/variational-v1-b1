# 万一 Edge 退出测试报告

测试覆盖：

- Long/Short 实际入场 Edge 的数量加权。
- 部分减仓保持剩余入场均价；减仓后再开仓只按剩余成本和新成交重算。
- 万一以下 `0.009%` 不触发，精确 `0.0100%` 触发。
- Short 使用当前 Long Edge，Long 使用当前 Short Edge。
- 历史实际平仓滑点不阻止下一次符合万一条件的下单。
- Short/Long 当前梯度档位最大仓位查询。
- 当前仓位到达档位目标后停止万一减仓。
- 普通梯度信号优先，万一信号不覆盖正常梯度目标。
- 普通梯度跨零、轮次结算与新方向轮次建立。
- 非零轮次持久化、重启恢复和仓位不一致停机。
- 下单日志区分“梯度阈值触发”和“万一 Edge 触发”。

关键边界：实际平仓平均只用于记录，不是下一笔发单门槛。每一笔万一退出只根据当时可执行反向 Edge 与实际入场均价之间是否达到 `0.0100%` 判断。
